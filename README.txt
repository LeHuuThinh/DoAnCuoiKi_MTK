// Thông tin
Project CoffeeOrderSystem được code bằng Java sử dụng IDE Intellij.

Sử dụng JDK 23

// Hướng dẫn thực thi file tự chạy
Để chạy file CoffeeOrderSystem.jar trong terminal cần:
- Folder Database (chứa restaurant.db và script.sql) và file CoffeeOrderSystem.jar nằm cùng trong một folder.
VD: FolderName -- Database
	       |_ CoffeeOrderSystem.jar
	       |_ ...
Sau đó trong thư mục FolderName, chuột phải Open in Terminal sau đó gõ lệnh: 
"C:\Users\ADMIN\.jdks\openjdk-23.0.1\bin\java -jar CoffeeOrderSystem.jar"
là sẽ chạy được.

Lưu ý: Đường dẫn C:\Users\ADMIN\.jdks\openjdk-23.0.1\bin\java là dẫn đến file chứa JDK 23 của máy, có thể sẽ khác nhau ở mỗi máy.

Hoặc đơn giản là mở project trong IDE IntelliJ và chạy.
